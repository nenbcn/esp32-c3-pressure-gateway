# ESP32-C3 Pressure Gateway for MICA

This project implements a modular FreeRTOS-based gateway for the MICA ecosystem using an ESP32-C3 and a WNK80MA pressure sensor.

**All coding standards, architecture, hardware mapping, and workflow rules are documented in:**

`.github/copilot-instructions.md`

Please refer to that file for all technical and development guidelines.

---

## About this Project

This repository contains the firmware for a pressure gateway designed for integration in the MICA ecosystem.  
It is intended for real-time pressure measurement, event detection, and secure cloud reporting.

- Uses PlatformIO for build and deployment.
- Modular codebase, designed for maintainability and scalability.
- Hardware and software details may evolve as the project develops.
