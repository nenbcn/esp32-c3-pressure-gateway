# Proyecto PlatformIO para ESP32-C3

Este directorio está destinado para archivos de header específicos del proyecto.

Los archivos de header del proyecto son aquellos que contienen declaraciones de funciones, 
constantes, y otras definiciones que serán utilizadas en múltiples archivos fuente (.cpp) 
del proyecto.

## Uso típico:
- Definiciones de constantes del proyecto
- Declaraciones de funciones globales
- Estructuras y tipos de datos personalizados
- Configuraciones específicas del hardware
